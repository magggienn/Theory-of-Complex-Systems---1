# Data Analysis and Visualization
This repository contains Jupyter notebooks for data analysis and visualization using Python.
The main libraries used are pandas, numpy, matplotlib, and seaborn.

## Requirements
- Python 3.x
- Jupyter Notebook
- pandas
- numpy
- matplotlib
- seaborn
